# Library Signs

A one-file web app that generates printable shelf signs for Lang Library.

## Use it

Open **`Lang Sign Maker.html`** in a browser (double-click it). No install, no internet required.

- **Sign type** — *Section* (letter tab) or *Title* (big banner)
- **Category** — Fiction, Comics, Non-Fiction, Young Reader (color changes automatically)
- **Section text** — type `A`, `AM-AZ`, `POEMS`, or `TRANSPORT BA-BZ`; the big letter and range split automatically
- **Title banners** — set the subtitle (`A-Z` or `Poems & Transport`) and shelf rack number
- **Print** — "Add to sheet" batches signs onto one Letter-landscape page; the printed date stamps itself automatically

Signs use the Proxima Nova font when installed, falling back to Montserrat.

Tip: if colors print blank, enable "Background graphics" in the browser's print dialog.
